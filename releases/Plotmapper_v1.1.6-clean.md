# PlotMapper v1.1.6-clean

Pre-release for the clean PlotMapper refactor.

## Highlights
- Clean modular implementation with generated standalone `index.html`.
- New canonical `.plotmap.json` project format for map, settings, and points.
- Markdown project import/export removed from the main UI; legacy conversion will be handled separately later.
- Reader remains available for linear route review, editing, jump-to-map, HTML export, and PDF print flow.
- PNG export, route drawing, helper points, labels, notes, locations, durations, filters, focus mode, undo/redo, themes, splash, tutorial, and info panel preserved and stabilized.

## Known Gaps
- Example screenshots/workflow images are still pending.
- Legacy Markdown/old JSON conversion is not included in this release.
- This is marked as a pre-release until more real-project smoke testing is complete.

## Asset
- `Plotmapper_v1.1.6-clean.html` is the standalone browser build.
