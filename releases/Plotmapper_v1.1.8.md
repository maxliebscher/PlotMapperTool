# PlotMapper v1.1.8

Feature release for route presentation and fog-of-war workflows.

## Highlights
- New Presentation Mode for revealing the current route step by step.
- Compact floating controls for previous, next, show all, reset, Focus Fog settings, and exit.
- Configurable Focus Fog with `Off`, `Focus`, and `Whole Route` modes.
- Live controls for surroundings visibility, current focus radius, path width, edge softness, and old path emphasis.
- Keyboard support for presenting routes: next, previous, reset, show all, and exit.
- README now includes a screenshot of the Focus Fog presentation view.
